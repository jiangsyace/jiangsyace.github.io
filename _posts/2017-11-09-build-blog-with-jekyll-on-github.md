## 使用模版

使用模版最简单的方式，就是直接把模版从git仓库down下来，然后修改_config.yml就行了


## Ruby命令
```sh
gem install jekyll
gem install bundle
gem install github-pages
bundle install github-pages
bundle install
jekyll serve
bundle exec jekyll serve
```
